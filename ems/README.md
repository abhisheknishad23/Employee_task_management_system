
# Employee Task Management System

A sleek, responsive Task Management System built to bridge the gap between management and staff.

# Features
- Role-Based Access: Distinct interfaces for Admins and Employees.
- Task Lifecycle: Manage tasks through "New," "Active," and "Completed" states.
- Data Persistence: Uses LocalStorage/Database to keep tasks saved.
- Responsive Design: Fully functional on mobile and desktop devices.

# Installation
1. Clone the repo:
   `git clone https://github.com/abhisheknishad23/Employee_task_management_system.git`
2. Install dependencies:
   `npm install`
3. Run the development server:
   `npm run dev`
